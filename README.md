## 📌 Overview

CivicFix is a modern web application that streamlines the process of reporting civic infrastructure issues such as potholes, garbage accumulation, water leakage, sewage problems, damaged street lights, and more.

The platform combines a clean user experience with future-ready AI capabilities to improve complaint management, enhance transparency, and support smarter urban governance.

---

##  Features

### Citizen Portal

* Report civic issues in a few clicks
* Upload images of the issue
* Select complaint category
* Add location and description
* Responsive interface for all devices
* Modern glassmorphism UI
* Smooth animations and interactions

### Dashboard

* Total complaint statistics
* Pending complaints
* In Progress complaints
* Resolved complaints
* Complaint cards
* Search and filters
* Analytics-ready design

##  Getting Started

### Clone the Repository

```bash
git clone https://github.com/sakshi-exe/Civic-Fix.git
```

### Navigate to the Project

```bash
cd Civic-Fix
```

### Run the Frontend

```bash
cd civicfix
python3 -m http.server 8000
```

Open your browser and visit:

```text
http://localhost:8000
```

---

## 🌐 Public Testing

Expose the local server using Cloudflare Tunnel:

```bash
cloudflared tunnel --url http://localhost:8000
```
